## 📖 Project Overview
The **Online FIR Management System** is a robust web application designed to bridge the gap between citizens and law enforcement. Built within **24 hours** during the Hack Horizon 2K25 hackathon, this platform allows users to file complaints securely and track their status in real-time.

Our goal was to eliminate the hassle of physical visits for minor complaints and ensure **24/7 accessibility** with verified security.

---

## 🚀 Key Features

### 🔐 Secure Authentication
- **Twilio OTP Integration:** Real-time mobile number verification to prevent spam and fake FIRs.
- **Secure Login:** Role-based access for Users and Admins.

### 📝 Complaint Handling
- **Digital Filing:** Easy-to-use forms for lodging FIRs with incident details.
- **Status Tracking:** Users can track if their case is *Pending*, *Under Investigation*, or *Resolved*.
- **History Logs:** Maintain a record of all past complaints.

### 💻 UI/UX & Support
- **Interactive Dashboard:** Clean and intuitive interface built with React.js.
- **24/7 Support System:** Instant help feature for urgent queries.
- **Responsive Design:** Fully functional on Mobile and Desktop devices.
